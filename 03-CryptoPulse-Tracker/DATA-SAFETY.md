# Seguridad de los Datos (Play Console) — CryptoPulse

| Pregunta | Respuesta |
|----------|-----------|
| ¿Recopila o comparte datos personales del usuario? | **No.** La app solo descarga datos de mercado públicos. |
| ¿Datos personales recopilados? | **Ninguno.** No hay cuentas ni perfiles. |
| ¿Usa internet? | **Sí**, para obtener precios de la API pública de CoinGecko. |
| ¿Se transmiten datos personales a terceros? | **No.** Solo se realizan peticiones de lectura de precios. |
| ¿Cifrado en tránsito? | **Sí (HTTPS).** |
| Identificadores / ubicación / analítica | **No se recopilan.** |
| ¿El usuario puede borrar datos? | La caché local se elimina al borrar los datos de la app o desinstalar. |

> Nota: aunque la app use internet, **no recopila datos personales**; conviene declararlo así con
> claridad. La conexión a CoinGecko puede implicar el registro de IP por su parte (propio de
> cualquier servicio web).
